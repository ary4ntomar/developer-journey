# PostgreSQL Authentication Practice

A practice project built while learning PostgreSQL and the `pg` library with Express.js.

## Overview

This project implements a simple authentication workflow using PostgreSQL as the database.

Features include:

* User signup
* User signin
* PostgreSQL integration
* SQL queries using the `pg` package
* JWT token generation
* Authentication middleware

## Concepts Practiced

### PostgreSQL

* Database connection using Neon
* SQL queries
* INSERT statements
* SELECT statements
* Returning inserted records

### Security Concepts

* Parameterized queries using placeholders (`$1`, `$2`, `$3`)
* SQL injection prevention
* JWT-based authentication

### Backend Development

* Express.js
* REST APIs
* Request validation
* Authentication flow

## Learning Outcome

This project helped me understand the difference between MongoDB/Mongoose workflows and PostgreSQL-based workflows while building a familiar authentication system using SQL.
