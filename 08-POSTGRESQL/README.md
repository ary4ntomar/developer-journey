# PostgreSQL

This section contains my transition from NoSQL databases (MongoDB) to relational databases using PostgreSQL.

The focus here is understanding relational data, SQL-based querying, table relationships, and integrating PostgreSQL with backend applications.

Topics explored include:

* SQL fundamentals
* PostgreSQL
* CRUD operations
* Relational database concepts
* Parameterized queries
* Authentication workflows
* Database integration with Express.js

This folder represents the beginning of my journey into SQL-based database systems.
