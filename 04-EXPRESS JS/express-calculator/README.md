# Express Calculator

A simple calculator application built with Express.js and a basic HTML frontend.

## Built With

- HTML
- JavaScript
- Node.js
- Express.js

## Concepts Practiced

- Express server setup
- GET and POST routes
- Route parameters
- JSON request body handling
- Middleware
- Serving static HTML with Express
- Frontend-to-backend communication using Fetch API

## Note

Created while learning how frontend code communicates with backend routes.
