# Password Hashing & Input Validation Practice

A backend practice file created while learning how to improve authentication safety using password hashing and request validation.

## Built With

* Node.js
* Express.js
* bcrypt
* Zod
* JWT

## Concepts Practiced

* Validating signup input with Zod
* Validating signin input with Zod
* Hashing passwords with bcrypt before storing them
* Comparing plain passwords with hashed passwords during signin
* Returning validation errors for incorrect input
* Generating JWT tokens after successful signin

## What This Practice Focuses On

The main purpose of this code is to understand two important backend concepts:

1. Passwords should not be stored directly.
2. User input should be validated before being processed.

## Current State

This is a learning practice file, not a complete production-ready authentication system.

It was created to understand how bcrypt and Zod fit into a basic signup/signin flow.
