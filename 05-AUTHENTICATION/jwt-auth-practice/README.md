# JWT Auth Practice

A practice project created while learning JWT-based authentication in Express.js.

## Built With

- HTML
- JavaScript
- Node.js
- Express.js
- JWT
- Axios

## Concepts Practiced

- User signup
- User signin
- JWT token generation
- JWT verification
- Protected routes
- Token-based API access
- LocalStorage
- Authenticated requests

## Current State

This project is part of my learning archive. It is not a complete application and may contain incomplete or experimental frontend functionality.
