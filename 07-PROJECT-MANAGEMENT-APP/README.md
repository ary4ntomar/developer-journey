# Project Management App

A full stack project management application built while learning backend development, databases, authentication, and application architecture.

This project combines multiple concepts learned throughout the journey into a single working application.

## Built With

- HTML
- JavaScript
- Axios
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT

## Features

- User authentication
- Organization management
- Member management
- Board creation
- Issue creation and assignment
- Issue status workflow
- Protected routes
- MongoDB database integration

## Concepts Practiced

- REST APIs
- Express routing
- Middleware
- JWT authentication
- Authorization
- MongoDB schemas and models
- Mongoose relationships
- CRUD operations
- Frontend and backend integration
- Application structure and architecture

## What I Learned

This project was my first attempt at building a larger application where multiple technologies worked together.

While building it, I learned how authentication, databases, APIs, frontend pages, and business logic connect to create a complete application.

## Current State

This project is part of my learning archive and reflects my understanding at the time it was built. It may contain rough implementations, unfinished improvements, or areas that could be refactored in the future.
