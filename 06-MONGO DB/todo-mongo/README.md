# Todo MongoDB

A backend practice project created while learning how to use MongoDB with Express.js.

## Built With

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT

## Concepts Practiced

- MongoDB connection
- Mongoose schemas
- Mongoose models
- User signup
- User signin
- JWT authentication
- Protected routes
- Creating todos
- Reading user-specific todos
- Deleting todos
- Connecting database records with users

## Current State

This project is part of my learning archive. It was created while learning MongoDB and backend authentication, so it may contain rough code, incomplete parts, or experimental implementation details.

## Important Note

Database credentials and secrets should not be committed publicly. Use environment variables for MongoDB connection strings and JWT secrets.
